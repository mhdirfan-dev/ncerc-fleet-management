About This System

Campus Fleet Management System

An AI-powered solution for real-time bus detection and fleet monitoring at NCERC campus gate.

Real-Time Detection

YOLOv12 AI model detects buses at the campus gate and reads Kerala number plates using EasyOCR with automatic error correction.

Schedule Monitoring

Automatically monitors morning (07:00–09:00) and evening (16:00–18:00) windows. Sends Telegram alerts for any missing buses.

Live Dashboard

Real-time web dashboard with Socket.IO push updates. View entry/exit logs, filter by date, search by bus number, and print reports.

Technology Stack

🤖 YOLOv12 👁️ EasyOCR 🍃 MongoDB ⚡ Flask 🔌 Socket.IO 🐍 Python 3.10 📷 OpenCV 🧠 PyTorch
